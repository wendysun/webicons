使用方法：

引入亿家净水基本样式表：@import:"url(base.css)"

例如调用购物车正常图标,使用短语元素标签span、i等引入样式：{{ <i class="gbicon gbicon-ac-cart"> <i/> }}

如果购物车动态切换效果，例如显示正常图标，选中后的图标样式：

  调用方法1：{{ <i class="gbicon gbicon-ac-cart-on"> </i>}} 或 {{ <i class="gbicon gbicon-ac-cart-ex"> </i> }}

  调用样式2：{{ <a href="" class="on"> <i class="gbicon gbicon-ac-cart"> </i> </a> }}

  调用方法3：{{ <a href="" class="ex"> <i class="gbicon gbicon-ac-cart"> </i> </a> }}
